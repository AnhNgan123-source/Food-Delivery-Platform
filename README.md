## Công nghệ sử dụng
- Backend: Spring Boot
- Frontend: React / Thymeleaf
- Database: PostgreSQL
- AI (nếu có): OpenAI API
## Cài đặt và chạy
### Yêu cầu
- Java 17+
- Node.js 18+ (nếu dùng React)
- PostgreSQL
### Chạy Backend
cd backend
./mvnw spring-boot:run
### Chạy Frontend (nếu dùng React)
cd frontend
npm install
npm start
### Truy cập
- Frontend: http://localhost:3000
- Backend API: http://localhost:8080
